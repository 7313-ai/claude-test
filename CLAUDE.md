# EV TINT 랜딩 프로젝트 가이드

이 저장소는 전기차 신차 패키지(썬팅·PPF 등) 서비스의 랜딩 페이지를 만드는 프로젝트입니다.
모든 작업(카피·콘텐츠·검수)은 아래 원칙을 따릅니다.

## 1. 브랜드 포지셔닝

- **핵심 메시지:** 투명한 신뢰로 만들어가는 전기차 신차 패키지.
- 저희는 과장 및 홍보성 허위 상업 멘트가 아닌, **진실된 서비스**를 제공합니다.
- **핵심 타깃:** 프리미엄 전기차를 보유한 여유 있는 오너층(고가 전기차 신차 오너).

## 2. 페이지 구조 (순서 고정)

히어로 → 문제 → 해결 → 특징 → 고객 FAQ → CTA

## 3. 말투·톤

- 간결하고 신뢰감 있게, **존댓말** 사용.
- 컨설팅 전문가 톤. 차분하고 단정적인 과장 없이 근거 중심으로.

## 4. 금지 사항

- 허위·과장 표현 금지.
- 근거 없는 수치(차단율·온도·실적 등) 금지. 수치를 쓸 경우 측정 기준·출처를 함께 표기.
- "반드시 성과·수익 보장" 류의 보장성 표현 금지.
- 고객사·고객 실명의 무단 사용 금지.

## 5. 필수 고지

- 페이지 **하단에 한 줄** 반드시 포함:
  〈자세한 맞춤 컨설팅은 1:1 문의해주세요.〉

## 6. 협업 방식 (에이전트 워크플로)

진행 순서:

1. **조사팀** — `positioning`(강점·타깃·한 줄 메시지) → `faq-writer`(고객 FAQ 정리)
2. **제작팀** — `copywriter`(섹션·카피·CTA 콘텐츠 초안 작성)
3. **검수** — `reviewer`(톤·과장·보장 표현·정보 누락 검수)

- **최종 산출물:** `index.html`
- reviewer 검수를 통과하지 못한 내용은 게시하지 않습니다.
