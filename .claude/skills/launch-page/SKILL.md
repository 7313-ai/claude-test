# Launch Page — 출시 랜딩 카피 작성 가이드

## 1. 언제 쓰는가

신규 제품·기능·이벤트를 알리는 **출시 랜딩 페이지의 카피를 작성하거나 다듬을 때** 사용합니다.

- 새 랜딩 페이지의 섹션·문구·CTA를 처음부터 작성할 때
- 기존 페이지를 출시 방침(포지셔닝·톤)에 맞게 다시 정비할 때
- 과장·근거 없는 수치를 걷어내고 신뢰 기반 카피로 교정할 때

> 사실 정보가 부족하면 추측해 채우지 말고, 무엇이 필요한지 질문한다. 조사팀(positioning, faq-writer) → 제작팀(copywriter) → reviewer 워크플로와 함께 쓰면 가장 좋다.

## 2. 섹션별 카피 공식

페이지는 아래 **6단계 순서를 고정**한다.

| 섹션 | 목적 | 카피 공식 |
|---|---|---|
| **히어로** | 첫인상·핵심 약속 | (한 줄 포지셔닝 메시지) + (타깃에게 주는 핵심 가치) + 1차 CTA. 근거 없는 수치·이벤트 압박 배제. |
| **문제** | 공감 | 타깃이 실제로 겪는 고민 2~3가지를 "~합니다/~어렵습니다" 톤으로 단정 없이 짚는다. |
| **해결** | 신뢰 | 그 문제를 어떻게 푸는지 원칙/방식 2~3가지. "왜 그렇게 하는지" 근거를 함께 제시. |
| **특징** | 차별점 | 핵심 차별점 3~4개를 근거와 함께. 효과는 "~에 도움을 줍니다" + 조건 단서로 표현. |
| **고객 FAQ** | 마지막 의문 해소 | 결제·신청 전 가장 많이 묻는 질문만. 답할 수 없는 건 "견적/문의 시 안내"로 정직하게. |
| **CTA** | 행동 | 신청 흐름(단계) + 무료/조건 명시 + 입력 폼 + 연락처. 마지막에 필수 고지 한 줄. |

### 카피 작성 원칙
- 말투: 간결하고 신뢰감 있게, **존댓말**, 컨설팅 전문가 톤.
- 한 섹션 = 하나의 메시지. 헤드라인은 결론부터.
- 모든 주장에는 근거(사실)를 붙인다. 근거 없으면 쓰지 않는다.

## 3. 작성 체크리스트

작성·정비 후 게시 전 아래를 모두 확인한다.

**금지 (하나라도 있으면 수정)**
- [ ] 허위·과장 표현 (최고/무조건/완벽/절대/100% 등)
- [ ] 근거 없는 수치 (차단율·온도·실적 등) — 쓰려면 "최대" 표기 + 측정 기준·출처 병기
- [ ] "반드시 성과·수익 보장" 류의 보장성 표현
- [ ] 고객사·고객 실명/로고의 무단 사용, 동의 없는 후기
- [ ] 가짜 카운트다운·할인 압박 등 기만적 연출

**필수 (반드시 포함)**
- [ ] 6단계 구조(히어로→문제→해결→특징→고객 FAQ→CTA) 순서 준수
- [ ] 페이지 하단에 필수 고지 한 줄 포함
- [ ] CTA에 신청 방법·비용 조건(무료 여부) 명시
- [ ] 연락처(전화/이메일) 노출
- [ ] 효과 주장에는 조건 단서("차종·조건에 따라 다름") 병기

## 4. 좋은 예시 1개 (히어로 섹션)

> **헤드라인:** 투명한 신뢰로 만들어가는 전기차 신차 패키지
> **서브카피:** 프리미엄 전기차의 가치에 맞는 썬팅·PPF·전동 선쉐이드를, 차량 스펙을 근거로 1:1 맞춤 제안해 드립니다.
> **본문:** 광고 문구가 아닌 차량 스펙을 근거로 말씀드립니다. 차종·연식·유리 면적·주행 환경을 함께 살펴 귀하의 전기차에 맞는 구성을 가입·결제 없이 무료로 안내해 드립니다.
> **CTA:** 무료 맞춤 견적 받기

**왜 좋은가**
- 한 줄 포지셔닝("투명한 신뢰")으로 시작해 타깃(프리미엄 전기차 오너)을 명확히 겨냥
- 근거 없는 수치(99%, -7℃ 등) 대신 "차량 스펙을 근거로"라는 검증 가능한 약속
- 과장·보장 표현 없이 컨설팅 톤의 존댓말 유지, CTA의 조건("가입·결제 없이 무료")을 정직하게 명시
